# MyTABATA
HIIT / Tabata Fitness Timer and Workout Generator

## Install

`composer install`

`vi .environment.php`

Include the following contents:

```
<?php
define('DB_HOST', 'localhost');
define('DB_USER', '');
define('DB_PASS', '');
define('DB_NAME', 'tabata');
define('DB_PREFIX', '');
```
